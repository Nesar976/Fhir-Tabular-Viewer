 FHIR Tabular Viewer (POC)

This is a small proof-of-concept built with React to display FHIR resources in a simple tabular format.  
The goal is to take raw FHIR JSON and present it in a way that’s easier to read and inspect.

 How to run

1. Install dependencies:
   npm install

2. Start the app:
   npm start

3. Open in browser:
   http://localhost:3000

 Notes

- This is an early POC, not a production-ready tool.
- The focus is on basic rendering and readability of FHIR data.
